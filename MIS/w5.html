Week 5 | IT Infrastructure

🔧 What is IT Infrastructure?

  • All hardware, software, networks, and services needed for an enterprise
  • Includes computing platforms, facilities, and IT services
  • Viewed as a "service platform" enabling business value

📜 Evolution of Infrastructure

  • Mainframe & Minicomputer era (1959–)
  • Personal Computer era (1981–)
  • Client/Server era (1983–)
  • Enterprise Computing era (1992–)
  • Cloud & Mobile Computing (2000–)

⚙️ Key Drivers of Evolution

  • Moore’s Law (doubling computing power)
  • Metcalfe’s Law (network value = number of members²)
  • Declining communication/storage costs
  • Technology standards = compatibility + scale

💻 IT Infrastructure Components

  • Hardware: Clients, Servers, Mainframes
  • OS: Windows Server, Unix, Linux, Chrome OS, iOS
  • Enterprise Software: SAP, Oracle
  • Data Management: SQL, DB2, MySQL, Hadoop
  • Networking: Cisco, Juniper, SKT, KT
  • Internet Platforms: Hosting, cabling, tools (Java, .NET)
  • Consulting & Integration: Legacy support and system integration

📈 Trends in Hardware Platforms

  • Mobile digital platforms: smartphones, tablets, wearables
  • Quantum computing: simultaneous processing power
  • Virtualization: hardware centralization and flexibility
  • Green computing: efficient, low-energy IT

☁️ Cloud Computing Types

  • IaaS: Infrastructure (e.g., AWS)
  • PaaS: Development platform (e.g., Azure)
  • SaaS: Software as a service (e.g., Dropbox)

📊 Platform Flexibility

  • Scalability and pay-as-you-go pricing
  • Outsourcing and hybrid cloud models
  • New policies required for mobile and cloud tools

📉 Investment Strategy

  • Balance investment – avoid over/under spending
  • Use competitive forces model to evaluate IT value
  • Consider IT strategy, market demand, and rival infrastructure
